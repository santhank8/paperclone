# Voice Rules: AI Detection Patterns

## Kill List (auto-fail any draft containing these)

### Openers
- "Great question!"
- "I'd be happy to help"
- "Absolutely!"
- "Here's the thing"
- "It's worth noting"
- "Let me explain"
- "To be fair"
- "Interestingly enough"
- "Fun fact:"

### Transitions
- "That being said"
- "With that in mind"
- "Having said that"
- "In other words"
- "At the end of the day"
- "Moving forward"
- "On the other hand"

### Buzzwords
- "leverage" (use: "use")
- "utilize" (use: "use")
- "streamline" (use: "simplify" or "speed up")
- "empower" (use: "let" or "enable")
- "unlock" (use: "get" or "enable")
- "harness" (use: "use")
- "elevate" (use: "improve")
- "robust" (use: "solid" or just describe what it does)
- "seamless" (use: "smooth" or describe why)
- "cutting-edge" (use: name the specific thing)
- "game-changer" (use: describe the actual change)
- "deep dive" (use: "look at" or "break down")

### Punctuation Patterns
- Em dashes anywhere
- More than 1 exclamation mark per post
- Ellipsis for dramatic effect ("And then... it worked.")
- ALL CAPS for more than one word

### Structure Tells
- Numbered lists (reads as LinkedIn)
- "1/" thread starts without a hook
- Posts that start with a question then answer it immediately
- "Did you know..." openers

## Safe Replacements

| AI Pattern | Human Alternative |
|-----------|-------------------|
| "I've found that X works well" | "X works" |
| "It's important to note that" | [just state the thing] |
| "This is a game-changer because" | [describe the actual impact] |
| "Here are 5 tips for" | Start with the best tip, no preamble |
| "In my experience" | [just state it as fact] |
