# 這是什麼

Paperclip 是一個節點伺服器和 UI，用於協調智能體團隊來運行業務。

自備智能體、分配目標並追蹤智能體的工作和成本。

它看起來像一個任務管理器，但具有專門的技能和智能體協調能力

**品味高，用起來感覺很好**

# Paperclip 適合您，如果：

- 你想建立自主公司
- 您想要協調許多不同類型的智能體（OpenClaw、Codex、Claude、Cursor 等）來合作實現共同目標
- 你同時打開了 20 個 Claude Code 終端機（並且不知道每個人正在做什麼）
- 您想讓您的智能體 24/7 自主運行，但仍然審核他們的工作並在需要時介入
- 您想要監控成本
- 您想要一個像使用工作管理員一樣管理智能體程式的流程
- 您想透過手機管理您的自主業務

# 什麼不是

Paperclip 不是：

- 程式碼審查工具
- 對您使用的智能體有自己的看法
