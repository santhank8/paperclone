# Paperclip

**Paperclip 是自主經濟的支柱。 **我們正在建構自主人工智慧公司運作的基礎設施。我們的目標是讓 Paperclip 驅動的公司共同創造與世界上最大國家的 GDP 相媲美的經濟產出。我們所做的每一個決定都應該服務：讓自主公司更有能力、更容易管理、更可擴展、更真實。

## 願景

自主公司——以真實結構、治理和問責制組織起來的人工智慧勞動力——將成為全球經濟的主要力量。沒有一家公司。數千。百萬。整個經濟層運作在人工智慧勞動力上，透過 Paperclip 進行協調。

Paperclip 不是該公司。 Paperclip 讓公司成為可能。我們是控制平面、神經系統、操作層。每個自治公司都需要結構、任務管理、成本控制、目標協調和人力治理。那就是我們。我們之於自主公司就像企業作業系統之於人類一樣──只不過這一次，作業系統才是真正的軟體，而不是隱喻。

衡量我們成功的標準不是一家公司是否運作。問題在於 Paperclip 是否成為自主公司賴以生存的預設基礎，以及這些公司是否集體成為與國家產出相提並論的重要經濟力量。

## 問題

任務管理軟體還不夠。當你的整個員工隊伍都是人工智慧智能體時，你需要的不僅僅是一個待辦事項清單——你需要一個適合整個公司的**控制平面**。

## 這是什麼

Paperclip 是一家人工智慧智能體公司的指揮、通訊和控制平面。這是您可以：

- **像員工一樣管理智能體** — 僱用、組織和追蹤誰做什麼
- **定義組織結構** — 智能體本身在其中運作的組織架構圖
- **即時追蹤工作** — 隨時查看每個智能體正在做什麼
- **控製成本** — 每個智能體的代幣工資預算、支出追蹤、燒錢率
- **與目標保持一致** - 座席人員了解他們的工作如何服務於更大的使命
- **儲存公司知識** — 組織的共享大腦

## 架構

兩層：

### 1.控制平面（本軟體）

中樞神經系統。管理：

- 智能體註冊和組織架構圖
- 任務分配和狀態
- 預算和代幣支出跟踪
- 公司知識庫
- 目標層次結構（公司→團隊→智能體→任務）
- 心跳監控－了解智能體何時處於活動狀態、空閒狀態或卡住狀態

### 2.執行服務（適配器）

智能體在外部運行並向控制平面報告。智能體只是啟動並運行的 Python 程式碼。適配器連接不同的執行環境：- **OpenClaw** — 初始適配器目標
- **心跳循環** - 簡單的自訂Python，循環、簽入、工作
- **其他** — 任何可以呼叫 API 的運行時

控制平面不運行智能體。它協調它們。特工們跑到哪裡就跑到哪裡，打電話回家。

## 核心原則

您應該能夠透過 Paperclip 一眼就了解您的整個公司 - 誰在做什麼、成本是多少以及是否有效。
