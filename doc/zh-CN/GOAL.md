# Paperclip

**Paperclip 是自主经济的支柱。**我们正在构建自主人工智能公司运行的基础设施。我们的目标是让 Paperclip 驱动的公司共同创造与世界上最大国家的 GDP 相媲美的经济产出。我们做出的每一个决定都应该服务于：让自主公司更有能力、更易于管理、更可扩展、更真实。

## 愿景

自主公司——以真实结构、治理和问责制组织起来的人工智能劳动力——将成为全球经济的主要力量。没有一家公司。数千。百万。整个经济层运行在人工智能劳动力上，通过 Paperclip 进行协调。

Paperclip 不是该公司。 Paperclip 使公司成为可能。我们是控制平面、神经系统、操作层。每个自治公司都需要结构、任务管理、成本控制、目标协调和人力治理。那就是我们。我们之于自主公司就像企业操作系统之于人类一样——只不过这一次，操作系统是真正的软件，而不是隐喻。

衡量我们成功的标准不是一家公司是否运作。问题在于 Paperclip 是否成为自主公司赖以生存的默认基础，以及这些公司是否集体成为与国家产出相媲美的重要经济力量。

## 问题

任务管理软件还不够。当你的整个员工队伍都是人工智能智能体时，你需要的不仅仅是一个待办事项列表——你需要一个适合整个公司的**控制平面**。

## 这是什么

Paperclip 是一家人工智能智能体公司的指挥、通信和控制平面。这是您可以：

- **像员工一样管理智能体** — 雇用、组织和跟踪谁做什么
- **定义组织结构** — 智能体本身在其中运作的组织架构图
- **实时跟踪工作** — 随时查看每个智能体正在做什么
- **控制成本** — 每个智能体的代币工资预算、支出跟踪、烧钱率
- **与目标保持一致** - 座席人员了解他们的工作如何服务于更大的使命
- **存储公司知识** — 组织的共享大脑

## 架构

两层：

### 1.控制平面（本软件）

中枢神经系统。管理：

- 智能体注册和组织架构图
- 任务分配和状态
- 预算和代币支出跟踪
- 公司知识库
- 目标层次结构（公司→团队→智能体→任务）
- 心跳监控——了解智能体何时处于活动状态、空闲状态或卡住状态

### 2.执行服务（适配器）

智能体在外部运行并向控制平面报告。智能体只是启动并运行的 Python 代码。适配器连接不同的执行环境：- **OpenClaw** — 初始适配器目标
- **心跳循环** - 简单的自定义Python，循环、签入、工作
- **其他** — 任何可以调用 API 的运行时

控制平面不运行智能体。它协调它们。特工们跑到哪里就跑到哪里，并打电话回家。

## 核心原则

您应该能够通过 Paperclip 一眼就了解您的整个公司 - 谁在做什么、成本是多少以及是否有效。
