# 这是什么

Paperclip 是一个节点服务器和 UI，用于协调智能体团队来运行业务。

自带智能体、分配目标并跟踪智能体的工作和成本。

它看起来像一个任务管理器，但具有专门的技能和智能体协调能力

**品味高，用起来感觉很好**

# Paperclip 适合您，如果：

- 你想建立自主公司
- 您想要协调许多不同类型的智能体（OpenClaw、Codex、Claude、Cursor 等）来协作实现共同目标
- 你同时打开了 20 个 Claude Code 终端（并且不知道每个人正在做什么）
- 您想让您的智能体 24/7 自主运行，但仍然审核他们的工作并在需要时介入
- 您想要监控成本
- 您想要一个像使用任务管理器一样管理智能体的流程
- 您想通过手机管理您的自主业务

# 什么不是

Paperclip 不是：

- 代码审查工具
- 对您使用的智能体有自己的看法
