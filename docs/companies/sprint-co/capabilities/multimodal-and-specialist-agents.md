# Multimodal Output and Specialist Agents Protocol

Purpose: operationalize roadmap tasks 167, 174, 175, and 176.

## 1. Multimodal Outputs (Task 167)

Allowed output modalities:
- Markdown narrative
- Diagrams (architecture and flow)
- Wireframe imagery
- Screenshot evidence

Rules:
- Every multimodal artifact must include a plain-text summary
- Non-text assets must be versioned and linkable from sprint artifacts

## 2. DevOps Agent Role (Task 174)

Primary responsibilities:
- CI/CD reliability
- Environment consistency
- Deployment safety controls
- Infrastructure configuration hygiene

Integration points:
- Works with Enforcer on release gates
- Works with QA on pre-release validation

## 3. Data Analyst Agent Role (Task 175)

Primary responsibilities:
- Analytics interpretation
- Trend anomaly explanation
- Dashboard quality and signal integrity

Integration points:
- Works with Stakeholder on usage outcomes
- Works with Historian on multi-sprint narratives

## 4. Growth Hacker Agent Role (Task 176)

Primary responsibilities:
- Conversion funnel optimization experiments
- Engagement hypothesis generation and validation
- Growth-focused experiment backlog management

Integration points:
- Works with Stakeholder on user impact priorities
- Works with Planner to sequence growth experiments

Output artifacts:
- Specialist recommendations appended to planning and board materials
