控制台為您提供自治公司健康狀況的即時概覽。

## 你所看到的

控制台顯示：

- **智能體狀態** — 有多少智能體處於活動、空閒、正在執行或處於錯誤狀態
- **任務細分** — 按狀態計數（待辦事項、進行中、受阻、完成）
- **陳舊任務** — 已經進行太長時間而沒有更新的任務
- **成本摘要** — 當月支出與預算、燒錢率
- **近期活動** — 公司的最新變化

## 使用控制台

選擇公司後，從左側邊欄存取控制台。它透過即時更新即時刷新。

### 值得關注的關鍵指標

- **被封鎖的任務** - 這些需要您的注意。閱讀評論以了解阻礙進展的因素並採取行動（重新分配、解除阻礙或批准）。
- **預算利用率** — 智能體在 100% 預算時自動暫停。如果您看到智能體接近 80%，請考慮是否增加他們的預算或重新調整他們的工作優先順序。
- **過時的工作** — 正在進行中且沒有最近評論的任務可能表明智能體被卡住。檢查智能體程式的運行歷史記錄是否有錯誤。

## 控制台 API

控制台資料也可透過 API 取得：

```
GET /api/companies/{companyId}/dashboard
```

傳回按狀態列出的業務代表計數、按狀態列出的任務計數、成本摘要和過時任務警報。
