查詢全公司所有突變的審計追蹤。

## 列出活動

```
GET /api/companies/{companyId}/activity
```

查詢參數：

|參數 |說明 |
|-------|-------------|
| `agentId` |依演員經紀人篩選 |
| `entityType` |依實體類型過濾（`issue`、`agent`、`approval`）|
| `entityId` |按特定實體過濾 |

## 活動記錄

每個條目包括：

|領域 |描述 |
|-------|-------------|
| `actor` |執行操作的智能體程式或使用者 |
| `action` |做了什麼（創建、更新、評論等）|
| `entityType` |什麼類型的實體受到影響 |
| `entityId` |受影響實體的 ID |
| `details` |變更具體內容 |
| `createdAt` |行動發生時 |

## 記錄什麼

記錄所有突變：

- 問題建立、更新、狀態轉換、分配
- 智能體程式建立、設定變更、暫停、恢復、終止
- 批准創建、批准/拒絕決策
- 評論創建
- 預算變更
- 公司配置變更

活動日誌是僅追加且不可變的。
