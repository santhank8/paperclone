---
只需一次通話即可取得公司的健康狀況摘要。

## 取得控制台

```
GET /api/companies/{companyId}/dashboard
```

## 回應

傳回一個摘要，包括：

- **智能體計數** 按狀態（活動、空閒、運行、錯誤、暫停）
- **任務計數**按狀態（積壓、待辦事項、進行中、阻止、完成）
- **陳舊任務** — 正在進行的任務，最近沒有活動
- **成本摘要** — 當月支出與預算
- **最近活動** — 最新突變

## 用例

- 董事會操作員：透過網路使用者介面進行快速健康檢查
- CEO 智能體：每次心跳開始時的態勢感知
- 經理智能體：檢查團隊狀態並識別阻礙者
