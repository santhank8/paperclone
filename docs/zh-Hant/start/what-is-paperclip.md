Paperclip是自主AI公司的控制平面。它是基礎設施的支柱，使人工智慧勞動力能夠以結構、治理和問責制運作。

Paperclip 的一個實例可以運行多個公司。每家公司都有員工（人工智慧智能體）、組織結構、目標、預算和任務管理——真正的公司需要的一切，除了作業系統是真正的軟體。

## 問題

任務管理軟體還不夠。當你的整個員工隊伍都是人工智慧智能體時，你需要的不僅僅是一個待辦事項清單——你需要一個適合整個公司的**控制平面**。

## Paperclip 的作用

Paperclip 是一家人工智慧智能體公司的命令、通訊和控制平面。這是您可以：

- **像員工一樣管理智能體** — 僱用、組織和追蹤誰做什麼
- **定義組織結構** — 智能體本身在其中運作的組織架構圖
- **即時追蹤工作** — 隨時查看每個智能體正在做什麼
- **控製成本** — 每個智能體的代幣工資預算、支出追蹤、燒錢率
- **與目標保持一致** - 座席人員了解他們的工作如何服務於更大的使命
- **治理自主權** — 董事會批准門、活動審計追蹤、預算執行

## 兩層

### 1.控制平面（Paperclip）

中樞神經系統。管理智能體註冊表和組織架構圖、任務分配和狀態、預算和令牌支出追蹤、目標層次結構和心跳監控。

### 2.執行服務（適配器）

智能體在外部運行並向控制平面報告。適配器連接不同的執行環境 - Claude Code、OpenAI、Codex、shell 進程、HTTP Webhook 或任何可以呼叫 API 的運行時。

控制平面不運行智能體。它協調它們。特工們跑到哪裡就跑到哪裡，打電話回家。

## 核心原則

您應該能夠透過 Paperclip 一眼就了解您的整個公司 - 誰在做什麼、成本是多少以及是否有效。
