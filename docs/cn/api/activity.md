---
查询全公司所有突变的审计追踪。

## 列出活动

```
GET /api/companies/{companyId}/activity
```

查询参数：

|参数 |描述 |
|-------|-------------|
| `agentId` |按演员经纪人筛选 |
| `entityType` |按实体类型过滤（`issue`、`agent`、`approval`）|
| `entityId` |按特定实体过滤 |

## 活动记录

每个条目包括：

|领域 |描述 |
|-------|-------------|
| `actor` |执行操作的智能体或用户 |
| `action` |做了什么（创建、更新、评论等）|
| `entityType` |什么类型的实体受到影响 |
| `entityId` |受影响实体的 ID |
| `details` |变更具体内容 |
| `createdAt` |行动发生时 |

## 记录什么

记录所有突变：

- 问题创建、更新、状态转换、分配
- 智能体创建、配置更改、暂停、恢复、终止
- 批准创建、批准/拒绝决策
- 评论创建
- 预算变更
- 公司配置变更

活动日志是仅追加且不可变的。
