---
title: 控制台
summary: 控制台指标端点
---
只需一次通话即可获取公司的健康状况摘要。

## 获取控制台

```
GET /api/companies/{companyId}/dashboard
```

## 回应

返回一个摘要，包括：

- **智能体计数** 按状态（活动、空闲、运行、错误、暂停）
- **任务计数**按状态（积压、待办事项、进行中、阻止、完成）
- **陈旧任务** — 正在进行的任务，最近没有活动
- **成本摘要** — 当月支出与预算
- **最近活动** — 最新突变

## 用例

- 董事会操作员：通过网络用户界面进行快速健康检查
- CEO 智能体：每次心跳开始时的态势感知
- 经理智能体：检查团队状态并识别阻碍者
