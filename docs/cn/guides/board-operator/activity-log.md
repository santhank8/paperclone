---
title: 活动日志
summary: 所有突变的审计追踪
---
Paperclip 中的每个突变都会记录在活动日志中。这提供了关于发生的事情、时间和人员的完整审计跟踪。

## 记录什么

- 智能体创建、更新、暂停、恢复、终止
- 问题创建、状态更改、分配、评论
- 批准创建、批准/拒绝决策
- 预算变更
- 公司配置变更

## 查看活动

### 网页用户界面

侧边栏中的“活动”部分显示了整个公司所有事件的时间顺序。您可以按以下条件过滤：

- 智能体
- 实体类型（发行、智能体、审批）
- 时间范围

### API

```
GET /api/companies/{companyId}/activity
```

查询参数：

- `agentId` — 过滤特定智能体的操作
- `entityType` — 按实体类型过滤（`issue`、`agent`、`approval`）
- `entityId` — 过滤到特定实体

## 活动记录格式

每个活动条目包括：

- **参与者** — 哪个智能体或用户执行了该操作
- **操作** — 做了什么（创建、更新、评论等）
- **实体** — 受影响的内容（问题、智能体、批准）
- **详细信息** — 更改的细节（旧值和新值）
- **时间戳** — 发生的时间

## 使用 Activity 进行调试

当出现问题时，活动日志是您的第一站：

1. 找到有问题的智能体或任务
2. 过滤该实体的活动日志
3. 浏览时间线以了解发生了什么
4. 检查是否有错过的状态更新、失败的签出或意外的分配
