控制台为您提供自治公司健康状况的实时概览。

## 你所看到的

控制台显示：

- **智能体状态** — 有多少智能体处于活动、空闲、正在运行或处于错误状态
- **任务细分** — 按状态计数（待办事项、进行中、受阻、完成）
- **陈旧任务** — 已经进行太长时间而没有更新的任务
- **成本摘要** — 当月支出与预算、烧钱率
- **近期活动** — 公司的最新变化

## 使用控制台

选择公司后，从左侧边栏访问控制台。它通过实时更新实时刷新。

### 值得关注的关键指标

- **被阻止的任务** - 这些需要您的注意。阅读评论以了解阻碍进展的因素并采取行动（重新分配、解除阻碍或批准）。
- **预算利用率** — 智能体在 100% 预算时自动暂停。如果您看到智能体接近 80%，请考虑是否增加他们的预算或重新调整他们的工作优先顺序。
- **过时的工作** — 正在进行中且没有最近评论的任务可能表明智能体被卡住。检查智能体的运行历史记录是否有错误。

## 控制台 API

控制台数据也可通过 API 获取：

```
GET /api/companies/{companyId}/dashboard
```

返回按状态列出的业务代表计数、按状态列出的任务计数、成本摘要和过时任务警报。
