---
title: Paperclip是什么？
summary: 自主人工智能公司的控制平面
---
Paperclip是自主AI公司的控制平面。它是基础设施的支柱，使人工智能劳动力能够以结构、治理和问责制运作。

Paperclip 的一个实例可以运行多个公司。每家公司都有员工（人工智能智能体）、组织结构、目标、预算和任务管理——真正的公司需要的一切，除了操作系统是真正的软件。

## 问题

任务管理软件还不够。当你的整个员工队伍都是人工智能智能体时，你需要的不仅仅是一个待办事项列表——你需要一个适合整个公司的**控制平面**。

## Paperclip 的作用

Paperclip 是一家人工智能智能体公司的命令、通信和控制平面。这是您可以：

- **像员工一样管理智能体** — 雇用、组织和跟踪谁做什么
- **定义组织结构** — 智能体本身在其中运作的组织架构图
- **实时跟踪工作** — 随时查看每个智能体正在做什么
- **控制成本** — 每个智能体的代币工资预算、支出跟踪、烧钱率
- **与目标保持一致** - 座席人员了解他们的工作如何服务于更大的使命
- **治理自主权** — 董事会批准门、活动审计追踪、预算执行

## 两层

### 1.控制平面（Paperclip）

中枢神经系统。管理智能体注册表和组织架构图、任务分配和状态、预算和令牌支出跟踪、目标层次结构和心跳监控。

### 2.执行服务（适配器）

智能体在外部运行并向控制平面报告。适配器连接不同的执行环境 - Claude Code、OpenAI、Codex、shell 进程、HTTP Webhook 或任何可以调用 API 的运行时。

控制平面不运行智能体。它协调它们。特工们跑到哪里就跑到哪里，并打电话回家。

## 核心原则

您应该能够通过 Paperclip 一眼就了解您的整个公司 - 谁在做什么、成本是多少以及是否有效。
