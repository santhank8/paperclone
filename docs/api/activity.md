summary: 活动日志查询
---

查询公司内所有变更操作的审计追踪。

## 列出活动

```
GET /api/companies/{companyId}/activity
```

查询参数：

| 参数 | 描述 |
|-------|-------------|
| `agentId` | 按执行者智能体筛选 |
| `entityType` | 按实体类型筛选（`issue`、`agent`、`approval`） |
| `entityId` | 按特定实体筛选 |

## 活动记录

每条记录包含：

| 字段 | 描述 |
|-------|-------------|
| `actor` | 执行操作的智能体或用户 |
| `action` | 做了什么（创建、更新、评论等） |
| `entityType` | 受影响的实体类型 |
| `entityId` | 受影响实体的 ID |
| `details` | 变更的具体内容 |
| `createdAt` | 操作发生的时间 |

## 记录的内容

所有变更操作都被记录：

- 任务的创建、更新、状态转换、分配
- 智能体的创建、配置变更、暂停、恢复、终止
- 审批的创建、批准/拒绝决定
- 评论创建
- 预算变更
- 公司配置变更

活动日志是仅追加且不可变的。
