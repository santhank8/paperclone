---
title: 活动
summary: 活动日志查询
---

查询全公司所有变更审计记录。

## 列出活动

```
GET /api/companies/{companyId}/activity
```

查询参数：

| 参数 | 描述 |
|-------|-------------|
| `agentId` | 按执行智能体过滤 |
| `entityType` | 按实体类型过滤 (`issue`、`agent`、`approval`) |
| `entityId` | 按特定实体过滤 |

## 活动记录

每个条目包括：

| 字段 | 描述 |
|-------|-------------|
| `actor` | 执行该操作的智能体或用户 |
| `action` | 执行了什么操作（已创建、已更新、已评论等） |
| `entityType` | 受到影响的实体类型 |
| `entityId` | 受影响实体的 ID |
| `details` | 变更的详细信息 |
| `createdAt` | 动作发生的时间 |

## 记录的内容

所有变更均会被记录：

- 问题的创建、更新、状态转换、分配
- 智能体创建、配置变更、暂停、恢复、终止
- 审批的创建、批准/拒绝决策
- 评论的创建
- 预算变更
- 公司配置变更

活动日志只能追加 (append-only) 且不可变。
