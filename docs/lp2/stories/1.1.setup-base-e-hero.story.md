## User Story

Como **visitante da landing page** (CTO, VP Engineering, founder),
quero **entender imediatamente o valor do AIOX em < 10 segundos**,
para que **eu saiba se devo continuar explorando ou não**.

## Acceptance Criteria

### AC-1: Estrutura base
- [ ] Arquivo `landing-v2/index.html` criado com HTML5 boilerplate válido
- [ ] `lang="pt-BR"` no `<html>`
- [ ] Design tokens CSS em `:root` (herdar da LP1: lime, dark, Geist)
- [ ] Google Fonts: Geist + Geist Mono com preconnect
- [ ] Meta tags: title, description, OG tags, canonical configurados

### AC-2: Navbar
- [ ] Navbar sticky no topo com backdrop-filter blur ao scrollar
- [ ] Logo "Paperclip × AIOX" à esquerda
- [ ] Links: Docs | GitHub | Discord no centro (desktop)
- [ ] Botão "Começar grátis" à direita (lime background)
- [ ] Mobile: hamburger menu funcional

### AC-3: Hero Section
- [ ] Eyebrow badge: "Open Source • Claude Code Native • Story-Driven"
- [ ] H1 com gradient text (branco → lime), mínimo 48px desktop, 32px mobile
- [ ] Subheadline em text-secondary, máximo 20 palavras
- [ ] CTA primário: "Começar gratuitamente →" (lime, bold)
- [ ] CTA secundário: "⭐ GitHub" (outline lime)
- [ ] Ambos CTAs visíveis above the fold em 1280px e 375px
- [ ] Hero visual: terminal animation com typewriter loop (4 linhas AIOX workflow)
- [ ] Trust badges abaixo dos CTAs

### AC-4: Performance (validado no QA Gate)
- [ ] LCP < 2.5s (mobile e desktop)
- [ ] CLS < 0.1
- [ ] Lighthouse Performance >= 90
