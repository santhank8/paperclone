## User Story

Como **visitante**, quero **ver resultados concretos, me identificar com uma persona e ser convidado a começar**,
para que **eu tome a decisão de signup**.

---

## Acceptance Criteria

### AC-1: Seção Outcomes
- [ ] 4 métricas de impacto em destaque (números grandes, lime)
- [ ] 3 benefit cards: Velocidade, Qualidade, Previsibilidade
- [ ] Cada card: ícone, título H3, descrição 2-3 linhas

### AC-2: Seção "Para quem é"
- [ ] 2 persona cards: CTO/VP Eng | Founder não-técnico
- [ ] Cada card: avatar emoji, título, badge, pain quote, 3 solution bullets
- [ ] Hover: scale(1.01), border lime

### AC-3: Social Proof
- [ ] Stats do projeto (GitHub stars hardcoded ou via API)
- [ ] 2-3 quote cards OU "Early Access" section como fallback
- [ ] Background diferenciado (bg-surface)

### AC-4: CTA Final
- [ ] Headline: "Seu time de IA começa hoje."
- [ ] Subheadline: 1 frase
- [ ] Botão grande (56px height) lime background, dark text
- [ ] Gradient radial lime-glow no background da seção

### AC-5: Footer
- [ ] 3 colunas: Brand | Links | Créditos
- [ ] Links: Docs, GitHub, Discord, Paperclip App
- [ ] Copyright: "© 2026 Paperclip × AIOX"
- [ ] Crédito meta: "Construído com AIOX Framework"
- [ ] Sem links quebrados

### AC-6: Página completa — QA Final
- [ ] Lighthouse Performance >= 90 (mobile e desktop)
- [ ] Sem erros no console
- [ ] Scroll completo sem CLS
- [ ] Todos os links funcionando
