## User Story

Como **visitante**, quero **entender como o AIOX funciona e quem são os agentes**,
para que **eu visualize como isso se aplicaria ao meu time**.

---

## Acceptance Criteria

### AC-1: Seção "Como funciona"
- [ ] 4 workflow steps com número, ícone, título (≤4 palavras), descrição (≤20 palavras)
- [ ] Connector visual entre steps (linha com seta, desktop)
- [ ] Stack vertical em mobile
- [ ] Responsive em 375px, 768px, 1280px

### AC-2: Stats Bar
- [ ] 4 métricas: 10+ agentes | 60+ componentes | 5 workflows | 100% story-driven
- [ ] Full-width, dark background
- [ ] Números em lime, labels em text-secondary

### AC-3: Agents Showcase
- [ ] 6 agent cards: @morgan, @river, @dex, @quinn, @gage, @aria
- [ ] Grid 3-col desktop, 2-col tablet, 1-col mobile
- [ ] Cada card: avatar emoji, @name em mono lime, role, capability (1-2 linhas)
- [ ] Hover: border lime, lime glow
