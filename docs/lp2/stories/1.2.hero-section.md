## Story

**As a** CTO ou founder visitando a LP2,
**I want** ver um hero section com headline focado em outcome de negócio e CTA claro,
**so that** eu entenda em 5 segundos o que Paperclip × AIOX faz por mim — sem precisar ler documentação técnica.

## Acceptance Criteria

1. Hero section implementada em `landing-v2/index.html` com id `#hero`
2. Headline principal: variação de "Ship 3x mais rápido com seu time de IA" — linguagem de resultado, não de produto
3. Sub-headline explicando o diferencial em 1-2 frases: outcome language, sem jargão técnico (sem "heartbeat", "SDC", "checkout")
4. CTA primário visível above-the-fold: botão lime com texto urgente ("Começar Agora — É Grátis" ou "Acesso Antecipado")
5. CTA secundário opcional: link de texto "Ver como funciona ↓" (scroll âncora para `#demo`)
6. Background dark cockpit: `--bg-base` com elemento visual diferenciado de LP1 (não usar terminal/CLI como feature visual principal)
7. Responsive: headline usa `clamp()` para tamanho fluido — `clamp(2.5rem, 6vw, 5rem)`
8. Acessibilidade: `<h1>` único na página, contraste WCAG AA mínimo

## Dev Notes

### Fonte: `docs/lp2/project-brief.md`

**Ângulo diferenciador:** LP1 vende "o que é" (plataforma + framework). LP2 vende "o que você ganha" (outcomes de negócio).

**Headline sugerida pelo Project Brief:** "Ship 3x mais rápido com seu time de IA" — pode adaptar mas manter foco em resultado tangível.

**Visual diferenciado:** O project brief especifica que a LP2 não deve usar terminal como visual principal. Sugestões:
- Cards de métricas de negócio ("2.5x mais PRs por sprint")
- Dashboard mockup simplificado
- Grid de agentes com papéis claros (SM, Dev, QA, DevOps)

**Paleta hero:**
```css
background: var(--bg-base); /* #0a0a0a */
/* Elemento de destaque: */
color: var(--bb-lime); /* #D1FF00 para destaques */
color: var(--text-primary); /* off-white para texto principal */
```

**Tipografia hero (da visual-spec):**
```css
h1 {
  font-family: var(--font-display); /* 'TASA Orbiter Display' */
  font-size: clamp(2.5rem, 6vw, 5rem);
  font-weight: 800;
  letter-spacing: -0.02em;
  line-height: 1;
}
```

**Audiência do ICP:** CTO/VP Eng em startup (Série A-B), 10-50 engenheiros, pressão de board para acelerar roadmap — esta é a persona primária da LP2.
