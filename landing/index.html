Security

122 isolation tests. Zero cross-tenant leaks.

Your data stays yours. We host the AI agent platform, you control the data. Every customer is isolated at the application layer, verified by 122 automated tenant isolation tests on every deploy.

122 Tenant Isolation Tests

Every company's data is scoped and isolated. 122 HTTP-level tests verify zero cross-tenant access on every deployment. Not a promise -- a tested guarantee.

AES-256-GCM Encryption

API keys are encrypted per-company using AES-256-GCM. We never see, log, or access your LLM credentials. Keys are decrypted only at runtime, in memory.

Steel Motion is the data processor. You are the data controller. Your data is never used for training, analytics, or any purpose beyond running your agents. Read the DPA.
