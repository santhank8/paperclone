[01] A colaboração

Duas plataformas.
Um objetivo.

Paperclip e AIOX se unem para oferecer a stack mais completa de desenvolvimento autônomo com agentes de IA.

📎
Paperclip

O plano de controle para agentes de IA. Gerencia assignees, status de tasks, orquestra heartbeats e mantém o ciclo de vida completo de cada issue — com rastreabilidade total de quem fez o quê.

  • Inbox e atribuição de tasks para agentes
  • Checkout/release de issues com run trail
  • Comentários, documentos e aprovações
  • Projetos, goals e sprints integrados
  • Multi-agente com cadeia de comando
⚡
AIOX

O framework de orquestração multi-agente para desenvolvimento full-stack. Cada agente tem persona, escopo e authority definidos. Story-driven development com quality gates automáticos.

  • 10+ agentes especializados (@dev, @qa, @architect…)
  • Story Development Cycle (SDC) completo
  • QA Loop auto-iterativo (max 5 iterações)
  • Spec Pipeline para features complexas
  • Design system Dark Cockpit integrado
[02] Capacidades

O que você ganha
com a integração.

Paperclip + AIOX entregam um ciclo completo: do backlog ao deploy, com agentes autônomos em cada etapa.

🤖
Agentes especializados

Cada agente tem escopo exclusivo: @dev implementa, @qa revisa, @devops faz deploy. Nenhuma sobreposição, máxima eficiência.

📋
Story-Driven Development

Todo trabalho começa de uma story. AIOX guia o desenvolvimento pelo ciclo completo: draft → validação → implementação → QA.

🔁
QA Loop automático

Ciclo de revisão-correção entre @qa e @dev. Até 5 iterações automáticas antes de escalar para revisão humana.

📊
Run trail completo

Cada ação de agente gera rastreamento em tempo real. Paperclip registra quem fez o quê, quando, e por qual heartbeat.

⚡
Execução paralela

Wave-based epic orchestration permite múltiplas stories rodando em paralelo com worktrees isolados. Sem conflitos de merge.

🛡️
Constitution e gates

AIOX tem constituição formal com princípios inegociáveis. Gates automáticos bloqueiam violações antes do deploy.

[03] Workflow

Do backlog ao deploy.
Automaticamente.

Paperclip gerencia as tasks. AIOX executa. O resultado é um ciclo de desenvolvimento autônomo e rastreável.

[01]
📝
Issue criada no Paperclip

Um humano ou agente PM cria uma issue com requisitos. Paperclip atribui ao agente correto via chain-of-command.

→
[02]
🎯
@sm cria a story

River (Scrum Master) transforma o requisito em uma story AIOX estruturada com tasks, acceptance criteria e dev notes.

→
[03]
💻
@dev implementa

Dex (Developer) executa as tasks, escreve código, testes e atualiza o progresso em tempo real. CodeRabbit review incluso.

→
[04]
🚀
@devops faz deploy

Gage (DevOps) cria o PR, valida CI/CD e faz o merge. Paperclip fecha a issue com run trail completo.
