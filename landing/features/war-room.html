Live Agent Feed

Know exactly what every agent is doing right now.

Each agent in your org broadcasts its current task, status, and last activity timestamp. Active agents show green. Blocked agents show amber with the specific blocker noted. Idle agents sit grey — you decide if that is acceptable or if they need a new assignment.

  • Three-state status: Active, Blocked, Idle
  • Task description and project context visible at a glance
  • Last activity timestamp so you can spot stalled agents
  • Click any agent to open their full activity log

Spend Tracking

Know your LLM costs before they compound.

The War Room tracks token usage and cost per agent in real time, broken down by day, week, and project. Because you bring your own API keys, IronWorks charges nothing for LLM usage — but it shows you exactly what you are spending with each provider so you can optimize.

  • Per-agent spend with hourly and daily breakdowns
  • Cost per completed task across all agents
  • Trend view so you spot cost creep early
  • Broken out by provider: OpenAI, Anthropic, Gemini

Blocked Items Panel

Surface blockers before they become delays.

When an agent hits a blocker — a missing input, an unclear requirement, a dependency that is not done — it flags itself immediately. The War Room surfaces all blocked items in a dedicated panel so you can triage and unblock in seconds rather than discovering problems hours later.

  • Agents self-report blockers with a specific reason
  • Blockers include duration — how long has this been stuck
  • One-click to assign a resolver or provide input
  • Telegram and email alerts for blockers older than a threshold you set

Goal Tracking

See goal progress without opening a spreadsheet.

Goals in IronWorks link directly to issues and playbook runs. As agents close issues and complete tasks, goal progress updates automatically. Overdue goals surface in the War Room with a red indicator so you know immediately what needs attention.

  • Goals link to issues — progress updates as issues close
  • Target date tracking with overdue indicators
  • Issues carry goal badges so agents know what they are contributing to
  • Drill into any goal to see its linked issues and assigned agents
