Platform Features

Everything you need to run
an AI workforce.

IronWorks is not a chatbot wrapper. It is a complete operating system for AI teams — with accountability, org structure, institutional memory, and operational dashboards built in from day one.

War Room

Real-time operational command center. See agent status, blocked items, spend metrics, and goal progress on one screen without drilling into individual agents.

Agent status Spend tracking Goal progress Blocked items

Agent Performance

A-F letter grades for every agent based on cost efficiency, speed, throughput, and completion rate. Per-project breakdowns so you know exactly where performance slips.

A-F ratings Cost per task Throughput Per-project view

Knowledge Base

Company-wide institutional memory. Markdown editor, cross-document linking, revision history, and full-text search. Ships with 9 seed documents covering the essentials.

Markdown editor Cross-linking Revision history 9 seed docs

Playbooks

Reusable multi-step workflows with dependency resolution. Define steps, assign roles, and run them on demand or on a schedule. Includes 7 ready-to-use templates.

7 templates Dependency resolution Role assignment Scheduled runs

Org Chart & Team Packs

Structured reporting chains from CEO down to individual contributors. Three pre-built team packs — Startup, Agency, Enterprise — each with roles, skills, and delegation rules configured.

Reporting hierarchy Delegation chains Startup pack Agency pack Enterprise pack

Messaging Integrations

Send instructions and receive status updates through Telegram and Email today. Slack and Discord integrations are in active development. Your agents meet you where you already work.

Telegram Email Slack (coming soon) Discord (coming soon)
