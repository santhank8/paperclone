# Plugins de Exemplo

Este diretório existe como base de referência e template para criação de novos plugins.

Os plugins first-party usados de verdade pelo produto vivem fora daqui, em caminhos reais dentro de `packages/plugins/`:

- `packages/plugins/company-pulse`
- `packages/plugins/workspace-explorer`
- `packages/plugins/central-operacoes`

Use os exemplos para estudar estrutura, manifest, worker, UI e testes. Não trate `packages/plugins/examples/*` como origem canônica dos plugins internos do produto.

## Plugins reais correspondentes

- `plugin-hello-world-example` → `packages/plugins/company-pulse`
- `plugin-file-browser-example` → `packages/plugins/workspace-explorer`
- `plugin-kitchen-sink-example` → `packages/plugins/central-operacoes`
