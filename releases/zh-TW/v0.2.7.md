# v0.2.7

> 發佈時間：2026-03-04

## 改進

- **載入彈性** — 設定精靈現在在環境測試失敗後繼續運行，而不是卡住。如果您的 Anthropic API 金鑰不起作用，您可以重試或清除它，然後繼續進行不同的設定。
- **Docker 入門流程** — Docker 煙霧測試的預設設定更清晰，並改進了 `npx` 入門運行期間的控制台指導。
- **技能中的問題搜尋** — Paperclip 技能參考現在記錄了用於按關鍵字查找問題的 `q=` 搜尋參數。

## 修復

- **Markdown 清單渲染** — 修正了清單標記（`-`、`*`）在編輯器和評論視圖中無法正確渲染的問題。
- **已存檔的公司已隱藏** — 新問題對話方塊不再在公司選擇器中顯示已存檔的公司。
- **嵌入式 Postgres 要求** — 伺服器現在在嵌入式 DB 模式下運行時正確需要 `embedded-postgres` 依賴項，從而防止啟動失敗。
