# v0.2.7

> 发布时间：2026-03-04

## 改进

- **载入弹性** — 设置向导现在在环境测试失败后继续运行，而不是卡住。如果您的 Anthropic API 密钥不起作用，您可以重试或清除它，然后继续进行不同的配置。
- **Docker 入门流程** — Docker 烟雾测试的默认设置更清晰，并改进了 `npx` 入门运行期间的控制台指导。
- **技能中的问题搜索** — Paperclip 技能参考现在记录了用于按关键字查找问题的 `q=` 搜索参数。

## 修复

- **Markdown 列表渲染** — 修复了列表标记（`-`、`*`）在编辑器和评论视图中无法正确渲染的问题。
- **已存档的公司已隐藏** — 新问题对话框不再在公司选择器中显示已存档的公司。
- **嵌入式 Postgres 要求** — 服务器现在在嵌入式 DB 模式下运行时正确需要 `embedded-postgres` 依赖项，从而防止启动失败。
