# Penclip

Penclip 是 Paperclip 的中文增强版，目标是打造面向中国用户的AI自治项目管理与智能体协同平台。

当前增强方向：

- 完整中文界面与中英双语切换
- 默认 `zh-CN` 本地化体验
- Windows 开发、构建与运行兼容
- Penclip 品牌与中文站点入口
- 为后续国产大模型接入预留基础设施

## 快速开始

```bash
npx paperclipai onboard --yes
```

或者手动启动：

```bash
git clone https://github.com/penclipai/paperclip.git
cd paperclip
pnpm install
pnpm dev
```

默认会启动：

- API: `http://localhost:3100`
- UI: `http://localhost:3100`
